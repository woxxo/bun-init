# Bun init

Create a set of necessary files (including package.json) for a new Bun project. Working with [Bun](https://github.com/oven-sh/bun) only.

### How to use

To run the package direct from GitHub:
```bash
bun x woxxo/bun-init
```
or from npmjs.com
```bash
bun x bun-init
```
